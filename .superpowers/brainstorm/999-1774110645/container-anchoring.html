How should container anchoring work?

When a node is dropped inside a container's bounds, how does the relationship form?

A

Auto-anchor on drop (Recommended)

When you drag a node and release it inside a container's expanded area, it automatically becomes a child. Drag it outside to unparent. Visual indicator (highlight) shows when you're hovering over a container during drag.

Pros

  • Most intuitive — matches Figma/PowerPoint behavior
  • Zero extra clicks
  • Visual feedback during drag

Cons

  • Accidental parenting possible
  • More complex drag logic
B

Manual anchor via context menu

Right-click a node → "Move into container" → pick which container. More explicit but slower.

Pros

  • No accidental parenting
  • Simple to implement

Cons

  • Breaks flow — too many clicks
  • Feels like a database, not a canvas
C

Hybrid: Auto-anchor + modifier key to prevent

Auto-anchor on drop (like A), but hold Alt during drag to prevent anchoring. Best of both worlds.

Pros

  • Intuitive default behavior
  • Escape hatch for power users

Cons

  • Discoverability of Alt key
